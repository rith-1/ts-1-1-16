Overview:
The command-line utility is used by agents to view ciphered files.

Usage:
java topsecret
    Displays a list of available files

java topsecret <file_number>
    Displays the contents of the specific file

java topsecret <file_number> <cipher code>
    Displays the contents of the specific file using the cipher code


Examples:
1. list all available files:
    java topsecret

    Output:
    01 filea.txt
    02 fileb.txt
    03 filec.txt

2. display file #2
    java topsecret 02

    Output:
    [contents of fileb.txt, deciphered using default cipher code]

3. display file #1 using cipher code
    java topsecret 02 5

    Output:
    [contents of filec.txt, deciphered using 5]

If the file number is invalid, there will be an error message and exit.
