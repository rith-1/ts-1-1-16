This program is responsible for the command line interface used by the user to run the program.

PSUEDOCODE
if args == 0
    call method to display all the files

else if args == 1
    call method to display the file at the index given by the users argument

else if args == 2
    call method to display the file and use the key given by the user's second arg


TESTS
if zero arguments, then should list the numbered files available
if one valid argument, then should open correct value at the index given by that argument
if two valid arguments, then should open a correct file and use the key given by user's second arg

if invalid arguments, then should reprompt the user
