This program is responsible for all file managment in the program.It provides the tools for file
reading, listing, and output.

1. Data location
All documents are expected to be in a folder called "data". If there is not a data folder,
in the working directory, an exception will be thrown.

2.Data types
Only .txt files are read. Other files are ignored since they don't work with the output
needed for the cipher.

Methods
1. getAvailableFiles()
   Return Type: List of strings
   Parameters: None

   Purpose:
   - Scans the data folder for all .txt files
   - Returns a list of filenames

   Behavior:
   -returns empty list if no .txt files found
   -returns empty list if the folder doesn't exit
2.readfile(String filename)
   Return type: string
   parameters: string filename of the file to be read

   Purpose-reads all content of file
   Behaviour:
   -makes a path with "data" and filename
   -returns content as string with separate lines
   -returns error if not found
